CSS 的单位

绝对单位

  • pt：磅, 1/72 inch
  • pica: 派卡, 1/6 inch

相对单位

  • px：像素，相对于显示设备
  • ex: 相对于当下 font-size 字母的高
  • em: 相对于当下 font-size 字母的宽
  • rem: 相对于浏览器设定字体的 em，等同于<boy> 的 em
  • vh: 相对浏览器可视区域的高度的 1/100
  • vw: 相对浏览器可视区域的宽度的 1/100
  • vmax：可视区域长边
  • vmin: 可视区域短边
  • %：相对基准，用在不同的 css property，有不同的 100%
    • padding: 父容器宽度
    • margin: 父容器宽度
    • width: 父容器宽度
    • height: 父容器高度

对于字体转换的建议

  • 使用px可以保持美感加上一定的精度
  • 单纯使用字体时，em 和 rem 可能有更好的效果
  • <body> 上不设置大小，而使用浏览器预设大小
  • @media(print) 设定的css，可以使用pt这种印刷单位

其他的一些建议

字间距使用 em，行高使用 ex

CSS Cascade 串联/层叠

Cascade 由 Specificity 和 Ruleset order 决定

Specificity

使用(a,b,c,d)的元组来计算，其中:

  • a 直接写在 HTML element 上的 style
  • b 出现 id 选择器的次数
  • c 出现 class 选择器的次数，出现 [attribute=value] 形式属性选择器的次数，出现伪类选择器的 :pesudoclasses
  • d 出现 :sudoelements，tagname 类型选择器的次数
